# Keychron K12 Pro — proponowany layout 60%

Legenda:

- `____` = brak akcji / puste
- `TRNS` = przepuszcza klawisz z warstwy bazowej
- `MO(1)` = `Nav`
- `MO(2)` = `Num`
- `MO(3)` = `Mouse`
- `MO(4)` = `Sym`
